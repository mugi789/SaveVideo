# How to Install and Usage

Open your terminal or termux (android)

```
git clone https://github.com/mugi789/SaveVideo
cd SaveVideo/
python3 savevid.py
```

### _ Tested and Works on Ubuntu 18.04.6 LTS x86_64 _
